# Better Deal - แอปเปรียบเทียบราคา 🏷️⚖️

เว็บแอปพลิเคชันที่ทันสมัย รวดเร็ว และใช้งานง่าย สำหรับเปรียบเทียบราคาเพื่อค้นหา "ดีลที่ดีที่สุด (Best Deal)" ระหว่างสินค้าหลายๆ แบบ ออกแบบมาเพื่อการใช้งานบนมือถือเป็นหลัก (Mobile-first) และเน้นความพรีเมียมของหน้าตาผู้ใช้งาน (UI)

## 🌟 ฟีเจอร์หลัก

- **เพิ่มรายการได้ไม่จำกัด:** เปรียบเทียบราคาและปริมาณของสินค้าได้หลายรายการพร้อมกัน ไม่จำกัดแค่ 2 ชิ้น
- **คำนวณอัตโนมัติ (Real-time):** คำนวณราคาต่อหน่วยให้ทันทีที่พิมพ์เสร็จ พร้อมหาว่าสินค้าชิ้นไหนคุ้มค่าที่สุด
- **ไฮไลท์สินค้าที่คุ้มที่สุด:** ระบบจะติดป้าย "✨ คุ้มที่สุด" และเน้นกรอบสีเขียวให้กับตัวเลือกที่ประหยัดเงินที่สุดโดยอัตโนมัติ
- **รองรับโหมดกลางคืน (Dark Mode):** เปลี่ยนธีมสว่างและมืดได้ไหลลื่นสุดๆ สามารถกดเปลี่ยนเอง หรือเปลี่ยนตามการตั้งค่าของระบบเครื่องก็ได้
- **Progressive Web App (PWA):** สามารถกด "เพิ่มลงในหน้าจอหลัก" (Add to Home Screen) ทั้งใน iOS และ Android เพื่อให้ใช้งานได้เหมือนแอปพลิเคชันปกติ และรองรับการทำงานแบบออฟไลน์
- **เหมาะกับมือถือ (Mobile Optimized):** ดีไซน์สะอาดตา ตอบสนองไว (Responsive) และแสดงผลได้สวยงามในทุกขนาดหน้าจอ
- **UI สวยงามพรีเมียม:** ตกแต่งสไตล์ Glassmorphism (กึ่งโปร่งใส), แอนิเมชันนุ่มนวล และการไล่สี (Gradient) ที่ออกแบบมาอย่างพิถีพิถันผ่าน Tailwind CSS

## 🚀 เทคโนโลยีที่ใช้

- **HTML5 และ CSS3**
- **Vanilla JavaScript** (เขียนโค้ดเพียวๆ ไม่ใช้ Framework หนักๆ ทำให้โหลดเร็วมาก!)
- **Tailwind CSS** (ดึงผ่าน CDN เพื่อการจัดสไตล์ที่รวดเร็ว)
- **Service Workers & Web App Manifest** (สำหรับการทำให้เป็น PWA)

## 📦 วิธีใช้งาน

1. **เปิดแอป:** เข้าสู่เว็บไซต์ (หรือเปิดไฟล์ `index.html` ในเครื่อง)
2. **กรอกข้อมูล:** ใส่ "ราคา" และ "ปริมาณ" ของสินค้าที่ต้องการเปรียบเทียบ ตั้งแต่ 2 ชิ้นขึ้นไป
3. **เปรียบเทียบ:** แอปจะคำนวณราคาต่อหน่วยให้คุณเห็นทันทีแบบ Real-time
4. **ค้นหาผู้ชนะ:** การ์ดสินค้าที่มีราคาต่อหน่วยต่ำสุด จะถูกไฮไลท์ด้วยป้าย "✨ คุ้มที่สุด" 
5. **เพิ่มรายการอีก:** หากจะเทียบมากกว่า 2 ยี่ห้อ ให้กดปุ่ม "เพิ่มรายการเปรียบเทียบ"
6. **เริ่มใหม่:** กดปุ่ม "ล้างข้อมูล" เพื่อลบข้อมูลที่กรอกไว้ทั้งหมด และเริ่มต้นคำนวณใหม่

## 📱 วิธีติดตั้งเป็นแอป (iOS / Android)

คุณสามารถติดตั้ง "Better Deal" ลงมือถือได้โดยตรง ไม่ต้องผ่าน App Store หรือ Play Store:

**สำหรับ iOS (ด้วย Safari):**
1. เปิดเว็บไซต์นี้ใน Safari
2. กดปุ่ม **แชร์ (Share)** มี่อยู่ด้านล่างจอ (ไอคอนสี่เหลี่ยมมีลูกศรชี้ขึ้น)
3. เลื่อนลงมาแล้วเลือก **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)**
4. กด **"เพิ่ม" (Add)** มุมขวาบน แอปจะไปโผล่ที่หน้าจอโฮมของคุณ!

**สำหรับ Android (ด้วย Chrome):**
1. เปิดเว็บไซต์นี้ใน Chrome
2. กดไอคอนจุด 3 จุด (⋮) ที่มุมขวาบน
3. เลือก **"ติดตั้งแอป" (Install app)** หรือ **"เพิ่มลงในหน้าจอหลัก" (Add to Home screen)**

## 🛠️ สำหรับนักพัฒนา (Local Development)

หากต้องการนำไปรันในเครื่องของคุณเอง:

1. โคลน Repository นี้:
   ```bash
   git clone https://github.com/yourusername/better-deal.git
   ```
2. เข้าไปในโฟลเดอร์:
   ```bash
   cd better-deal
   ```
3. เปิดไฟล์ `index.html` บนเบราว์เซอร์
   *(หมายเหตุ: ส่วนของ Service Workers เพื่อให้รันออฟไลน์แบบ PWA ได้ อาจจะต้องรันผ่าน Local Server เช่น ใช้ปลั๊กอิน Live Server ใน VS Code หรือคำสั่ง `python -m http.server`)*
